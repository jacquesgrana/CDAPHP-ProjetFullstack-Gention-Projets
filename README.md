## CDA PHP - Projet Gestion de Projets

### Installation

- Dans un terminal ouvert à la racine du projet :

```bash
composer init
```

### Base de données

- dbname : 'gestion_projet'
- dbhost : 'localhost'
- dbuser : 'admin'
- dbpass : 'admin'
- sinon, modifier le fichier : 'src/Configuration/Config.php'

### Lancement

- Dans un terminal ouvert à la racine du projet :

```bash
php -S localhost:3000
```
